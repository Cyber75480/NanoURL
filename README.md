# 🔍 NanoURL Pro - IP Logger

A simple and powerful IP Logger tool made for educational and security awareness purposes.  
This tool helps you understand how IP tracking works and how connections are established over the internet.

---

## ⚙️ Features
- Fast IP logging
- Simple and clean interface
- Lightweight script
- Works with Cloudflare links
- URL masking support

---

## 🚀 Usage

```bash
git clone https://github.com/Cyber75480/NanoURL.git
cd NanoURL
chmod +x nanourl.sh
bash nanourl.sh
```
